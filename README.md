## Get Started
1. Install all the package
    ```sh
    npm install
    ```
2. Create a database
    ```sh
    npx sequelize db:create
    ```
3. Doing Migration
    ```sh
    npx sequelize db:create
    ```
4. Seeding all
    ```sh
    npx sequelize db:seed:all
    ```
